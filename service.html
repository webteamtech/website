Email

Voor "Call to Action" is een emailadres op website vermelden een van de mogelijkheden voor interacties tussen bezoeker en Business.

Vermelding

De eenvoudigste variant is alleen vermelden van emailadres zoals 1e plaatje hierboven. Bezoeker kan dan copy & paste dit adres en zelf besluiten hoe te gebruiken in mailApp of officeApp.

Start email App

Door in de Code toevoegen van [href=mailto:] kan bezoeker op het emailadres klikken waarna als overlay zijn haar eigen default emailApp verschijnt met concept bericht zoals 2e plaatje hierboven rechts. Een boodschap kan worden ingetikt en verzonden.

AVG Compliancy

In beide hierboven beschreven gevallen, omdat bezoeker op eigen device een App activeert kan website er van uitgaan dat bezoeker zelf voldoet aan AVG Privacywet, dus vanuit website bezien geen speciale SSL security maatregelen nodig.

Contactformulier

Voor "Call to Action" kan in website een Contactformulier worden opgenomen. Bezoeker kan een boodschap intikken en verzenden. Het bericht is technisch gezien een email. Omdat dit wordt gefaciliteerd door website zelf, zijn enkele zaken van toepassing.

Opties

  • Back-End Email Service
  • AVG compliant SSL
  • reCAPTCHA Spam Bot Protect

Email is zo normaal gebruikelijk dat weinige bedenken hoe zeer complex de technisch infrastructurele organisatie is. Het opzetten en beheren van een email Server lijkt alleen weggelegd voor Big Corp of die hard techneut.

Email Service

Beter is een email Service en het inregelen daarvan is dan ook een onderdeel bij opnemen van een Contactformulier. Een goede email Service Provider verziet ook in een AVG compliant SSL certificaat voor encryptie & security van de verbinding.

Spam

Ter voorkoming van ongewenste geautomatiseerde spam bots kan bij Contactformulier in website een reCAPTCHA filter worden opgenomen. Inbouwen kan gelijk bij oplevering van website of later als dat blijkt nodig te zijn.

Opties

  • Geen filter
  • Checkbox alleen afvinken I'am not a Robot
  • Checkbox met follow-up intikken reCAPCTHA

Live Chat

Voor "Call to Action" kan in website een Live Chat worden opgenomen. Verschijning van chat widget is meestal rechtsonder. Bij activering start een overlay voor heen-en-weer tekst intikken communicatie tussen bezoeker en Customer Service.

Opties

  • Back-End Chat Service
  • Mobile Contact Centre App
  • AVG compliant SSL
  • Widget locatie on screen instelbaar
  • Widget timing direct of conditioneel
  • Features customizable
  • Chat verslag via Email
