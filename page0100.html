Build & Service Web presents

Built on HTML5 & CSS3

Om op het internet een webpagina te bekijken kan gebruiker kiezen uit verschillende Browsers zoals Mozilla Firefox, Google Chrome, Safari, Microsoft Edge, Opera en nog veel meer.
En alle browsers zijn geprogrammeerd om computercodes HTML, CSS en Javascript te interpreteren naar acties op een digitaal beeldscherm.

Conventies omtrent inzetten van die onderliggende opmaaktaal HTML5 en stylesheet CSS3 wordt mondiaal aangestuurd door het W3C met de bedoeling dat, ongeacht keuze van browser, het resultaat op scherm gelijk is.

Responsive Web Design

Webdesign voor optimale webervaring voor een breed scala aan apparaten van desktop computerschermen tot mobiele telefoons. Een responsive website schaalt mee met de afmetingen van een scherm, zonder in te leveren op leesbaarheid van tekst of bruikbaarheid van de gebruikersinterface.

Flexibele grids zorgen ervoor dat breedtes en hoogtes van elementen in relatieve eenheden. Afbeeldingen worden ook percentueel geschaald zodat ze proportioneel worden weergegeven.
